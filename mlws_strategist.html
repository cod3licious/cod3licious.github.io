ML for Data Strategists Workshop (1 day)

AI: Realizing Opportunities

This course is aimed at employees, who have an interest in artificial intelligence and machine learning, and managers, who want to supervise data science projects.

  • You will get an overview of the topic of artificial intelligence / machine learning and typical fields of application.
  • You will get to know the data analysis process and understand which problems can be solved with machine learning and where difficulties can arise.

The course is based on my book A Practitioner's Guide to Machine Learning, which you can refer to for more details.

Prerequisites

  • Good command of the English language (all materials are in English)
  • Basic understanding of math (at high school level)
  • Have fun puzzling and an interest in new technologies
  • Complete the Preparation part before the first group session (about 20min; see below)

Agenda

The course takes place remotely via Microsoft Teams, Google Meet, Zoom, Slack, or similar. Please join the calls with your camera turned on so the sessions feel a bit more personal.

Important: Please complete the Preparation part before we meet for the course!


Preparation before the course (~20min)

  • Read the first chapter of the book: "Introduction"
  • Think about where in your daily life you're already using machine learning (in addition to the provided examples)

Course (9:00-17:00)

Welcome:
  • Introduction & Orga
  • Discuss where we're using ML
Lecture:
  • What is ML?
  • How do machines “learn”?
  • ML use cases
Coffee Break & Exercise:
  • Prepare a 90-second Spotlight presentation for one of the given ML use cases
  • Spotlight presentations
Lunch Break (ca. 12-13:00)
Lecture:
  • Solving problems with ML
Coffee Break & Exercise (Part 1):
  • Identify opportunities how ML could be used to improve a given process
Lecture:
  • Garbage in, Garbage out!
  • Data & Concept Drifts
  • Models can discriminate
  • Adversarial Attacks
Exercise (Part 2):
  • Assess the identified ML solutions with respect to different risks
Good Bye:
  • Conclusion
  • Feedback
