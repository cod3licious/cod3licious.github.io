ML in Practice Workshop

This course is aimed at practitioners, who want to analyze data with machine learning algorithms.

The three-day workshop follows a "flipped classroom" format, where the participants study the theory and solve the exercises at their own speed and the results are then discussed in the group sessions.

Learning Outcomes

  • Understand the fundamentals of machine learning.
  • Overview of the most important supervised learning algorithms and what needs to be considered when using them.
  • First practical experience in the application of machine learning algorithms by working through a case study on your own.
  • Basic knowledge of the central tools of the Python data science ecosystem (numpy, scipy, matplotlib, pandas, scikit-learn).
Please Note: This workshop focuses on the fundamental machine learning algorithms to solve practical problems in industry contexts, e.g., using sensor data. It does not cover deep learning models to work with, for example, images or text data.

Prerequisites

  • Good command of the English language (all materials are in English)
  • Math skills at university entrance level (especially linear algebra: matrices & vectors, functions)
  • Good programming and basic Python skills (e.g., by working through the supplied Python tutorial before the course)
  • Intrinsically motivated to learn new things
  • Desirable: First practical experience analyzing data
  • Technical requirements: Python installation on your own computer or access to a cloud environment, where you can run Jupyter notebooks (e.g., Google Colab)
  • Complete the Preparation part before the first group session (about 3 hours; see below)

Agenda

The course consists of several group sessions, where we discuss questions and results, and self-study parts in between, where you read through some chapters of the book and solve the programming exercises.

Important: Please complete the Preparation part before the first group session!

The group sessions take place remotely via Microsoft Teams, Google Meet, Zoom, Slack, or similar. Please join the calls with your camera turned on so the sessions feel a bit more personal.


Preparation before the course (~3 hours)

  • Read the first chapter of the book: "Introduction" (take notes if you have any questions!)
  • Answer Quiz 1 (also available as a pdf in case you can't access Google Forms)
  • Answer Quiz 2 [pdf]
  • Read the chapter: "ML with Python"
  • Complete the Python tutorial

Day 1

1st Group Session (~30 min):

  • Introduction
  • Discuss any questions you might have about the preparation part
  • Recommended: Determine groups for the work on the case study

Self-Study Part 1: Data

Block 1.1:
  • Read the whole chapter: "Data & Preprocessing"
  • Work through the first part of the Notebook: analyze toy dataset (Exploratory Data Analysis)
  • Explore the data for the case study in the Notebook: predicting hard drive failures and prepare some insights to present to the group. Feel free to work in pairs, especially if you're still new to Python.
Block 1.2:
  • Read the chapter "Supervised Learning Basics"
  • Answer Quiz 3 [pdf]

Day 2

2nd Group Session (~1 hour):

  • Discuss any questions you might have
  • Presentation of insights about the case study data

Self-Study Part 2: Supervised Learning

Block 2.1:
  • Read the chapter "Supervised Learning Models"
  • In parallel, work through the respective sections of the Notebook: supervised comparison
Block 2.2:
  • Work through the remainder of the Notebook: analyze toy dataset
  • Have a look at the cheat sheet, which includes a summary of the most important steps when developing a machine learning solution, incl. code snippets
  • Train your own models in the Notebook: predicting hard drive failures

Day 3

3rd Group Session (~20 min):

  • Discuss any questions you might have

Self-Study Part 3: Avoiding Common Pitfalls

Block 3.1:
  • Read the chapter: "Avoiding Common Pitfalls"
Block 3.2:
  • Finish the Notebook: predicting hard drive failures and prepare a presentation of your final results

4th Group Session (~1 hour):

  • Discuss any questions you might have
  • Presentation of the case study results

Outlook

If you want to learn more:
  • Read the chapters "Unsupervised Learning" and "Advanced Topics"
  • Have a look at the remaining programming exercises
  • Check out the section "Additional Resources"
  • Do an online course, e.g., the Coursera Deep Learning Specialization (by Andrew Ng)
